## Spyglass Connect

スマートフォンを[Spyglass Connect](https://github.com/beryndil/Spyglass-Connect)デスクトップコンパニオンとペアリングして、ローカルWiFi経由でMinecraftのワールドデータにアクセスできます。クラウドサーバーもアカウントも不要 ── すべてあなたのネットワーク内で完結します。

| 機能 | 説明 |
|------|------|
| **QRペアリング** | QRコードをスキャンして接続 ── 同じWiFiにいれば自動で再接続します |
| **キャラクタービューワー** | 装備の全体像を確認 ── 防具、武器、ツール、エンチャント |
| **インベントリビューワー** | インベントリ、防具、オフハンド、エンダーチェストの内容をすべて閲覧 |
| **チェストファインダー** | すべてのコンテナからアイテムを検索 ── チェスト、樽、シュルカーボックス、ホッパー |
| **構造物ロケーター** | 村、寺院、海底神殿、要塞などを座標と距離付きで検索 |
| **俯瞰マップ** | 構造物マーカーとプレイヤー位置を表示するインタラクティブな地形マップ |

完全暗号化(ECDH + AES-256-GCM)。mDNSによる自動再接続。
プロトコルv2でバージョンネゴシエーション対応 ── 互換性のないバージョンでは明確な更新メッセージが表示されます。

[Connectの詳細を見る](/connect/)

## グローバル検索

すべてのカテゴリを同時にリアルタイム検索。結果はタイプ別にグループ化され、関連性でランク付けされます。どの結果もタップすれば詳細ページに直接移動します。

## テーマ

15の内蔵テーマからアプリの外観をカスタマイズ ── ライト、ダーク、AMOLED、Minecraftにインスパイアされたカラースキームなどを含みます。

## オフラインファースト

すべてのゲームデータはアプリに同梱されており、インターネット接続なしで動作します。オンライン時は、バックグラウンド同期が12時間ごとにデータ更新を確認し、変更された分だけをダウンロードします。

## データ同期

Spyglass Connectは[Spyglass-Data](https://github.com/beryndil/Spyglass-Data)リポジトリと自動的に同期します。マニフェストがファイルごとのバージョンを追跡するため、変更されたデータのみがダウンロードされ、大きなダウンロードなしでアプリを最新の状態に保ちます。
