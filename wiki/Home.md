# Spyglass Wiki

**Your Minecraft world, on your phone.**

Spyglass is an Android companion app for Minecraft Java Edition 1.21.4. It combines a complete offline game database, 19 calculators, and a real-time desktop companion that streams your world data to your phone over local WiFi.

## Requirements

- Android 8.0+ (Oreo, API 26)
- ~30 MB storage
- Textures download ~1.3 MB on first launch
- WiFi connection for Spyglass Connect features
