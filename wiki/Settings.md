# Settings

Access Settings from the top-right menu on any screen. Settings are organized into sections provided by the app's module system.

## Appearance

| Setting | Options | Default |
|---------|---------|---------|
| Theme | 17 themes (see [Themes](Themes)) | Obsidian |
| Dynamic Color | On/Off (Android 12+ only) | Off |
| High Contrast | On/Off | Off |
| Font Scale | Small, Default, Large, Extra Large | Default |
| Haptic Feedback | On/Off | On |
| Reduce Animations | On/Off | Off |
| Startup Screen | Home, Browse, Calculators, Search | Home |

## Game Version

| Setting | Options | Default |
|---------|---------|---------|
| Minecraft Edition | Java / Bedrock | Java |
| Minecraft Version | Latest or specific version | Latest |
| Version Filter Mode | Show All, Highlight Unavailable, Hide Unavailable | Show All |
| Hide Unobtainable Blocks | On/Off | Off |
| Show Experimental Features | On/Off | Off |

## Browse

| Setting | Options | Default |
|---------|---------|---------|
| Default Browse Tab | Any of the 13 categories | Blocks |
| Show Favorites on Home | On/Off | Off |
| Show Tip of the Day | On/Off | On |
| Manage Favorites | Clear all favorites | — |

## Tools

| Setting | Options | Default |

## Data & Sync

| Setting | Options | Default |
|---------|---------|---------|
| Offline Mode | On/Off | Off |
| Sync Frequency | 1h, 6h, 12h, 24h | 12h |
| Sync Now | Manual trigger | — |
| Textures | Download / Update / Delete | — |

When you tap **Sync Now**, the app immediately checks the CDN for data updates and downloads any changed files. An inline progress indicator shows sync status.
